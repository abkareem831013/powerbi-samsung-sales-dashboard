# 📊 Samsung Galaxy Sales Dashboard (Power BI)

## 📌 Overview

This project analyzes Samsung Galaxy S-Series sales data (2020–2025) and provides insights into revenue trends, regional performance, and future forecasting.

## 🚀 Features

* KPI Metrics: Total Revenue, Total Units, YoY Growth
* Regional Sales Analysis
* Model-wise Performance
* Time-Series Trend Analysis
* 3-Year Forecast (2026–2028)

## 🛠 Tools Used

* Power BI
* DAX
* Power Query
* Data Modeling (Star Schema)
