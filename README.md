📊 Samsung Galaxy Sales Dashboard (Power BI)

📌 Project Overview

This project analyzes Samsung Galaxy S-Series sales data (2020–2025) and provides insights into revenue trends, regional performance, and future sales predictions.

📊 Key Features

* KPI metrics: Total Revenue, Total Units, YoY Growth
* Regional performance analysis
* Model-wise sales comparison
* Time-series analysis with monthly trends
* 3-year forecast (2026–2028)

 🔧 Tools Used

* Power BI
* DAX (Data Analysis Expressions)
* Data Modeling (Star Schema)
* Power Query
